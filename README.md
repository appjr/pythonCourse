## ğŸ› ï¸ Installation & Setup

### Option 1: Anaconda (Recommended for Beginners)

1. **Download Anaconda**: Visit [anaconda.com](https://www.anaconda.com/download) and download for your OS
2. **Install Anaconda**: Follow the installation wizard
3. **Launch Jupyter Notebook**: 
   - Open Anaconda Navigator
   - Click on "Jupyter Notebook"
   - Or use terminal: `jupyter notebook`

### Option 2: Python + Pip

1. **Install Python**: Download from [python.org](https://www.python.org/downloads/) (Python 3.10+)
2. **Install Jupyter**:
   ```bash
   pip install jupyter notebook
   ```
3. **Install required packages**:
   ```bash
   pip install -r requirements.txt
   ```

### Verify Installation

Open a terminal and run:
```bash
python --version
jupyter --version
```
